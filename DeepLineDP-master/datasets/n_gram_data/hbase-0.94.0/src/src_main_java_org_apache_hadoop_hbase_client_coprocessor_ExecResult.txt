* </p>
*
* @see exec

* @see org.apache.hadoop.hbase.client.htable#coprocessorexec(class, byte[], byte[], org.apache.hadoop.hbase.client.coprocessor.batch.call, org.apache.hadoop.hbase.client.coprocessor.batch.callback)
*/
